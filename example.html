jQuery.flickrPhotostream

On this page you'll find a few examples that demonstrate the jQuery.flickrPhotostream plugin

Plugin defaults

$.fn.flickrPhotostream.defaults = {
  userId: undefined,        // flickr userId.
  useSmallPhotos: true,     // use small images instead of medium sized images.
  photoCount: 6,            // number of photo's to show.
  callback: function() {}   // function that will be called after the latest tweet has been filled in.
};

Example 1: default plugin behaviour

Flickr PhotoStream

Code

$(function() {
  $( '#example1 div' ).flickrPhotostream({
    userId: '51313069@N02'
  });
});

      Flickr PhotoStream callback example

        Code

        $(function() {
  $( '#example4start' ).click(function(e) {
    e.preventDefault();
    $( '#log' ).text( 'start getting flickr photos' );
    $( '#example4 ul' ).flickrPhotostream({
      userId: '52386368@N00',
      photoCount: 4,
      callback: function() {
        $( '#log' ).text( 'finished!' );
      }
    });
  });
});
